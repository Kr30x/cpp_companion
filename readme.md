# cpp_companion

## Установка Linus/OSX

0) ```git clone https://github.com/Kr30x/cpp_companion.git ~/cpp_companion```

1) ```cd ~/cpp_companion```

2) ```python3 install.py```

3) Следуйте указаниям в консоли


## Использование 

0) ```cd ~/cpp_companion```

1) ```python3 test.py```

2) Следуйте указаниям в консоли
